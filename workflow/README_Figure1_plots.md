Figure1 plots use a fixed 2x3 species layout shared by:

- `classical_models`
- `network_methods`
- `gnn_models`

Top row:

- `fgraminearum_newlabel` -> `F. graminearum`
- `fgraminearum_oldlabel` -> `F. graminearum (old label)`
- `scerevisiae` -> `S. cerevisiae`

Bottom row:

- `human` -> `H. sapiens`
- `celegans` -> `C. elegans`
- `dmelanogaster` -> `D. melanogaster`

This panel order must not be auto-rearranged based on data availability.
`fgraminearum_oldlabel` and `dmelanogaster` must remain visible in their fixed panel positions.
