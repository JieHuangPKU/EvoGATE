### Critical finding: Figure2a results predate model runner code

- `run_frozen_protocol_model.py` modified **May 1**
- Figure2a results timestamped **April 12**
- Gap: **19 days**. The model runner was changed AFTER Figure2a was produced.
- Any regeneration of Figure2a would use different code → potential result drift.

### Provenance gap: Universal

All 284+ jobs across all workflows report "missing provenance/metadata". Snakemake metadata directory (`.snakemake/metadata/`) is empty. This means:
- Current Snakemake session has no record of what code/config/data produced each output
- Snakemake cannot auto-detect if a code change should trigger rerun
- Manual checksum comparison is the only available verification method

### Risk assessment

| Risk | Severity | Evidence |
|------|----------|----------|
| Stale results (old code → current output) | **HIGH** | Figure2a predates model runner by 19 days; 284 jobs no provenance |
| Mixed code versions across Figures | **HIGH** | Figure2a (Apr 12 code), Figure2b (Apr 11 code), Figure4/5 (May code) |
| Cannot reproduce exact results | **HIGH** | No environment lock + missing outputs/ + no provenance |
